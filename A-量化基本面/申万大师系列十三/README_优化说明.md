# 罗伯·瑞克超额现金流选股法则 - 优化版本

## 概述

本项目对原始的申万大师系列十三进行了全面优化，将依赖jqdata的代码转换为使用免费数据源（tushare + akshare），并大幅提升了代码的可读性和可维护性。

## 主要改进

### 1. 数据源替换
- **原版**: 依赖jqdata付费接口
- **优化版**: 完全使用tushare免费接口
- **优势**: 
  - 无需付费即可使用
  - 数据获取更加稳定
  - 支持更多数据类型

### 2. 代码架构重构
- **原版**: 函数式编程，代码分散
- **优化版**: 面向对象设计，模块化架构
- **改进**:
  - `DataProvider`类：封装所有数据获取逻辑
  - `RobertRickStrategy`类：实现选股策略
  - `StrategyAnalyzer`类：提供分析和可视化工具

### 3. 错误处理和日志
- 添加完善的异常处理机制
- 集成日志系统，便于调试和监控
- 数据获取失败时的优雅降级

### 4. 性能优化
- 分批获取数据，避免接口频率限制
- 智能重试机制
- 内存使用优化

### 5. 用户体验提升
- 清晰的进度提示
- 详细的使用文档
- 丰富的可视化分析工具

## 选股标准

罗伯·瑞克超额现金流选股法则的五个标准：

1. **市净率 < 3**: 确保估值合理
2. **股息率 > 市场平均**: 保证分红收益
3. **市盈率 < 市场平均**: 避免高估值陷阱
4. **负债率 < 33%**: 确保财务健康
5. **价现比 < 市场平均80%**: 重视现金流质量

## 注意事项

1. **数据延迟**: 财务数据存在发布延迟，建议使用已确认发布的财报期间
2. **接口限制**: tushare免费用户有调用频率限制，建议升级为积分用户
3. **数据质量**: 部分股票可能存在数据缺失，策略会自动过滤
4. **投资风险**: 策略结果仅供参考，实际投资需要综合考虑多种因素
