# FFScore因子计算 - 安装使用指南

## 项目概述

本项目将原始的聚宽平台FFScore因子计算代码修改为使用tushare和akshare数据源，实现了相同的功能但不再依赖聚宽平台。

## 环境要求

- Python 3.7+
- 稳定的网络连接（用于API调用）
- tushare账户和token（推荐积分版本）

## 安装步骤

### 1. 克隆或下载项目

```bash
# 如果是git仓库
git clone <repository_url>
cd A-量化基本面/华泰FFScore

# 或直接下载文件到本地目录
```

### 2. 创建虚拟环境（推荐）

```bash
# 使用conda
conda create -n fscore python=3.8
conda activate fscore

# 或使用venv
python -m venv fscore_env
source fscore_env/bin/activate  # Linux/Mac
# fscore_env\Scripts\activate  # Windows
```

### 3. 安装依赖包

```bash
pip install -r requirements.txt
```

### 4. 配置tushare token

#### 方法1：在代码中直接设置（已配置）
代码中已经设置了token：
```python
ts.set_token('ce9f8c37a4af987f6328321ed4b3f9379f695d6d6bdc5b59d454e3ad')
```

#### 方法2：环境变量设置
```bash
export TUSHARE_TOKEN='your_token_here'
```

#### 方法3：配置文件
在用户目录下创建`.tushare`文件夹并添加token配置。

## 快速开始

### 1. 基本使用

```python
from FFScore_tushare import FScore, get_trade_period

# 创建FScore实例
fscore = FScore()

# 设置股票池和日期
stocks = ['000001.XSHE', '000002.XSHE', '600000.XSHG']
date = dt.date(2023, 12, 31)

# 计算FScore
fscore.calc(stocks, date)

# 查看结果
print("FScore结果:", fscore.fscore)
print("基础指标:", fscore.basic)
```

### 2. 批量计算

```python
# 获取时间序列
periods = get_trade_period('2023-01-01', '2023-12-31', 'Q')  # 季度末

results = []
for period in periods:
    fscore.calc(stocks, period)
    if not fscore.fscore.empty:
        result_df = pd.DataFrame({
            'date': period,
            'code': fscore.fscore.index,
            'fscore': fscore.fscore.values
        })
        results.append(result_df)

# 合并结果
final_results = pd.concat(results, ignore_index=True)
```

### 3. 运行示例脚本

```bash
python example_usage.py
```

## 注意事项

### API限制

1. **tushare调用频率**
   - 免费版：200次/分钟
   - 积分版：更高频率
   - 建议使用积分版本

2. **数据延时**
   - 免费版可能有数据延时
   - 实时数据需要付费版本

3. **批量处理**
   - 大量股票计算时注意API限制
   - 建议添加适当延时

### 数据质量

1. **财务数据缺失**
   - 新上市股票可能缺少历史数据
   - ST股票数据可能不完整
   - 已在代码中添加处理逻辑

2. **数据一致性**
   - 不同数据源格式可能略有差异
   - 建议在使用前验证数据质量

### 性能优化

1. **缓存机制**
   - 当前版本未实现数据缓存
   - 建议在生产环境中添加

2. **并行处理**
   - 可以考虑多线程处理
   - 注意API调用限制

## 故障排除

### 常见问题

1. **tushare token错误**
   ```
   错误：Token验证失败
   解决：检查token是否正确设置
   ```

2. **网络连接问题**
   ```
   错误：连接超时
   解决：检查网络连接，可能需要代理
   ```

3. **数据获取失败**
   ```
   错误：获取财务数据失败
   解决：检查股票代码格式，确认数据可用性
   ```

4. **依赖包问题**
   ```
   错误：ModuleNotFoundError
   解决：重新安装requirements.txt中的包
   ```
